# Play market app info async parser.

## Installation:

### 1. install python3 (**VERSION 3.9+ IS NEEDED**): 
https://www.python.org/

### 2. clone project:

```shell
git clone git@gitlab.com:Tanat77/app-parser.git
cd app-parser
```

### 3. install requirements:

```shell
pip install -R requirements.txt
```
